# 🎯 Decision Lab — Frontend

Aplicación SPA para registrar decisiones y visualizar métricas de impacto personal.  
Parte del ecosistema **Decision Lab**.

---

## 🖥 Stack

- **Framework:** Vue 3 (Composition API) + Vite
- **Estado global:** Pinia
- **Ruteo:** Vue Router
- **Visualización:** Chart.js + vue-chartjs
- **HTTP:** Axios
- **Estilos:** Tailwind CSS v4
- **Testing:** Vitest + Vue Test Utils (próximamente)

---

## ✅ Requisitos

- Node.js 18+
- Backend de Decision Lab corriendo

---
